⚡ How This Works

Your Secrets Are Hashed Locally: When you type, your input is hashed in your browser using SHA-256. The server never sees your plaintext.

Constant-Time Comparison: Inside the enclave, a branchless bitwise algorithm compares the hashes in constant time—meaning execution speed reveals nothing about the result.

Server Blindness: The host system receives only encrypted payloads and a 1-bit result. It cannot observe timing variations, branch patterns, or memory access traces.

// Branchless constant-time comparison
uint32_t diff = hashA ^ hashB;
uint32_t match = ((diff | -diff) >> 31) ^ 1;
